# TOON Extension Docs

Basic documentation for the TOON VS Code extension.

## Quick Reference

- **File extension**: `.toon`
- **Format**: Indentation-based like YAML
- **Comments**: Start with `#`
- **Arrays**: Use `[item1, item2]` or multi-line format

## Example

```toon
name: Example
items: [a, b, c]
nested:
  key: value
```
